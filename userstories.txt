EPIC: Account & Authentication
1) As a new user, I want to sign up with email/password so that I can create a personal account.
2) As a returning user, I want to sign in securely so that I can access my tasks.
3) As a user, I want to reset my password so that I can regain access if I forget it.

EPIC: Core Task Management
4) As a user, I want to create a task with a title and notes so that I can remember what to do.
5) As a user, I want to edit a task’s title and notes so that I can update details later.
6) As a user, I want to delete a task so that I can remove items I no longer need.
7) As a user, I want to mark a task as completed so that I can track what’s done.
8) As a user, I want to undo a recent delete so that I can recover mistakes.

EPIC: Lists / Organization
9) As a user, I want to create multiple lists/projects so that I can group related tasks.
10) As a user, I want to reorder tasks within a list so that I can prioritize visually.
11) As a user, I want to add tags/labels to tasks so that I can categorize across lists.

EPIC: Dates & Reminders
12) As a user, I want to set due dates and times so that I know when tasks are expected.
13) As a user, I want reminders before due time so that I don’t miss deadlines.
14) As a user, I want recurring tasks (daily/weekly/custom) so that repeating work is automated.

EPIC: Search & Filters
15) As a user, I want to search tasks by keyword so that I can find items quickly.
16) As a user, I want to filter by status (all/active/completed) so that I can focus on what matters.

EPIC: Collaboration
17) As a user, I want to share a list with specific people so that we can work on tasks together.
18) As a user, I want to assign tasks to collaborators so that ownership is clear.
